### 📝 Descripción del Proyecto: GeekRadar

*¿Qué es y qué hace la app?*
Es una aplicación web full-stack interactiva y geolocalizada. Funciona como un radar inteligente que detecta la ubicación en tiempo real del usuario y, mediante Inteligencia Artificial, recomienda lugares, eventos y establecimientos específicos según la temática elegida (arte, cultura, tecnología, zonas gamer, etc.).

*¿Quiénes son los clientes y usuarios?*
* *Usuarios (B2C):* Turistas y comunidades nicho que buscan experiencias específicas en su ubicación actual.
* *Clientes (B2B):* Comercios locales, restaurantes temáticos y espacios culturales que pagan por posicionamiento dentro del radar de usuarios calificados.

*Stack Tecnológico:*
* *Frontend:* HTML5, CSS3, JavaScript.
* *Backend:* Python.
* *IA:* Modelos de lenguaje para recomendaciones personaliza

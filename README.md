es una aplicación web full stack interactiva diseñada para resolver el clásico dilema urbano: ¿A dónde ir a parchar cuando se tiene hambre y ganas de jugar? La plataforma funciona como una guía inteligente y geolocalizada que conecta a la comunidad geek y de videojuegos con restaurantes, arcades, cafés de juegos de mesa y zonas gamer de la ciudad
